water crisis in Kenya, built as part of CSC 317 - Web Software Development at San Francisco State University.
Contributors
Davis Oduor
Joonsung Oh

The project explores the Kenyan water crisis through six key topics:
Access — millions lack reliable, clean water
Sanitation — limited access to basic sanitation facilities
Contamination — waterborne diseases and pollution
Environment — arid land and climate-driven droughts
Urbanization — rapid city growth outpacing infrastructure
Solutions — grassroots and technology-driven responses
